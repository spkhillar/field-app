Admin page (Administration -> User Management)

This page is accessible only for users with admin role and it provides functionality to create/maintain users for Helios Field Application. On this page user can see the list of users with all the information other than password. Sorting is implemented on all the columns and user can also provide number of records he want to see per page from drop down given at bottom of page. Other buttons (functions) at the bottom of page are:

+ Filter

Filter allows you to manage the list of information on grid. To create a filter:

1. Click the filter button in your page. A window that allows you to specify your search criteria will appear.

2. Enter your search criteria. If you want to check that your search worked correctly, click the Find button.

3. You can provide multiple search criteria by using (+) sign appears in the pop-up. Click the Find button and refined list based on your search criteria will appear in grid.

4. If you want to remove a search criteria then use (-) sign appearing next to criteria.

5. Use Reset to remove all search criteria.

+ Refresh

A refresh or reload will cause the browser to re-check with the web server to make sure it has the latest copy of the web page you are viewing. To refresh the grid:

1. Click the Refresh button in your page. Grid data will be reloaded from the server to ensure that you are viewing the latest copy.

+ Add

Add allow you to add more users to use the application. To add a user:

1. Click the add button in your page. A window that allows you to provide user information will appear.

2. Enter Username, Password, First name, Last name, Phone, Email, Enable and Role information.

3. Click Submit - a new user is created and now he can use the application.

4. To crosscheck, verify if user name appears in the grid.

5. User name should be unique for each user.

+ Edit

Edit allow you to edit user information. To edit a user:

1. Select the user first from the grid and click the edit button in your page. A window that allows you to edit user information will appear. This window will have all existing information.

2. Override one or multiple information i.e. Password, First name, Last name, Phone, Email, Enable or Role.

3. Click Submit - user information is edited now.

4. To crosscheck verify from the grid. Other than password all the information can be seen here.

+ Delete

Delete allows you to delete users from the system. To delete a user:

1. Select the user first from the grid and click the delete button in your page. A confirmation window will appear.

2. Based on your selection user will be permanently deleted from the system.

+ Export

Export allow you to export the data in Excel format on user system. If filter is applied then only filtered records from the screen will be exported to excel. To export:

1. Click the export button in your page. A window will appear to save or open in Excel.

2. Based on your selection file will be saved on system.
