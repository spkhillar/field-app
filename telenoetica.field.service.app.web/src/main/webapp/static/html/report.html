This section contains 6 pages :

1. Routine Visit Report: This page contains list of all routine visits till date with filter, refresh and export to excel functions.

2. Callout Visit Report: This page contains list of all Callout Visit till date with filter, refresh and export to excel functions.

3. Diesel Visit Report: This page contains list of all Diesel Visit till date with filter, refresh and export to excel functions.

4. Maintenance Visit Report: This page contains list of all Maintenance Visit till date with filter, refresh and export to excel functions.

5. Diesel Details reports: This page contains list of Diesel Details reports generated in last 12 months and link to download the same. Diesel Details reports are generated on month end and send via email to configured mailing list.

6. Spares Utilization reports: This page contains list of Diesel Details reports generated in last 12 months and link to download the same. Spares Utilization reports are generated on month end and send via email to configured mailing list.
