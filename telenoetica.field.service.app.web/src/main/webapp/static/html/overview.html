Welcome to the one-stop source for help on HTN Field Application. Find everything you need for setting up, exploring features and usage information.

To download HTN Field App:

1. If a previous version is installed, make sure all visit data has been sent by using Send to Server

2. Uninstall the previous version of the application

3. Click on DOWNLOAD ANDROID APP link on top navigation. Download and install the application on your android phone or tablet device.

If you are experiencing any difficulties in installation, please contact HTN I.T Helpdesk.

You may or may not have access to all the pages of web application depending on your user role. If you have any specific need to access pages which are not accessible to you, please contact HTN I.T Helpdesk.

Android application can be used only by Field Service Engineers (FSE) and Admin users. Other users cannot login to android application on smart phones.
